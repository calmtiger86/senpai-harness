# Completion Evidence

여러 프로젝트의 완료 증거 현황을 한눈에 모아 보여주는 곳입니다.

프로젝트별 상세 증거는 `10_Projects/{프로젝트 이름}/Verification.md`와 각 프로젝트의 Completion Evidence 기록에서 확인하세요.

## 여기서 확인하는 것

- 어떤 프로젝트가 "구현 완료, 검증 전" 상태인지
- 어떤 프로젝트가 "검증 완료" 상태인지
- 증거가 부족해 "부분 완료"로 남아있는 항목

## 기억할 것

파일이 바뀌었다고 완료가 아닙니다. 완료 증거가 있어야 완료입니다. 증거가 부족하면 "부분 완료"라고 말하는 것이 원칙입니다.
