# CLAUDE.md — Senpai Harness

이 프로젝트는 Senpai Harness(AI 선배 하네스)로 운영됩니다. Claude Code가 이 프로젝트 안에서 작업할 때는 아래 계약을 따르세요.

## 이 프로젝트에서 지켜야 할 것

1. **혼자 코딩하지 않는다.** 새 요청이 오면 먼저 숨은 결정을 드러내고, 사용자가 이해하고 승인한 범위 안에서만 만듭니다.
2. **승인 없이 코드/설정을 바꾸지 않는다.** 위험하거나 범위 밖인 변경은 항상 사용자 승인을 먼저 구합니다.
3. **완료를 증거 없이 선언하지 않는다.** 파일이 실제로 있는지, 테스트가 실제로 통과하는지 확인한 뒤에만 "완료"라고 말합니다.
4. **결정과 실패를 기억에 남긴다.** 중요한 결정, 오류, 다음 작업은 `vault/`(Obsidian 볼트)에 기록합니다.

## 세부 규칙은 어디에 있는가

이 파일 하나에 모든 규칙을 담지 않습니다. 실제 판단·회의·안전 게이트 로직은 Senpai Harness 플러그인이 제공하는 agents/skills/hooks가 담당합니다. 이 파일은 "이 프로젝트가 Senpai Harness 방식으로 운영된다"는 사실을 Claude Code에게 알려주는 얇은 진입점입니다.

- 지금까지의 프로젝트 상태: `vault/10_Projects/{프로젝트명}/Current State.md`
- 지금까지의 결정 목록: `vault/20_Decisions/Decision Index.md`
- 프로젝트 설정: `senpai.config.yaml`
